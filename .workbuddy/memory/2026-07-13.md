# 2026-07-13 · Date-Tool 液态玻璃 UI 重构

## 做了什么
应用户要求，把 Date-Tool 整个 UI 从"AI 味"的暗色蓝紫霓虹风，重构为 **Verdigris 液态玻璃**风格。

**配色三原色**：`#1F4A48`（深青墨绿）/ `#C9A87C`（古铜金）/ `#F5F1EB`（米白）

## 三个交付物
1. **`DESIGN.md`**（24KB · 9 章节完整规范）—— 设计系统档案，给未来 AI 编程代理读
2. **`app/globals.css`**（18KB · 重写）—— **关键：零 .tsx 改动就能升级整个 UI**。所有现有组件（Uploader / DatasetList / ChartCard / InsightPanel / DataTable）都用 globals.css 里的类名（card / btn / btn-primary / dropzone / ds-item / badge / chart-card / banner / narrative / table-wrap / spinner 等），重写变量值和玻璃效果即可。
3. **`DESIGN-PREVIEW.html`**（24KB · 自包含预览）—— 不启 dev server 直接浏览器打开就能看效果，含三个 Tab：首页 / Dashboard / 组件实验室

## 设计要点（核心心智模型）
- **背景**：径向渐变（`radial-gradient(ellipse 1400px 900px at 30% 0%, #2D5F5C 0%, #1F4A48 45%, #0F2A29 100%)`）+ 右上角金色暖光点 模拟博物馆射灯
- **玻璃面**：所有容器加 `backdrop-filter: blur(24px) saturate(180%)` + `rgba(245,241,235,0.07)` 半透明 + `inset 0 1px 0 rgba(245,241,235,0.18)` 顶部高光 + 多层深绿阴影（不是纯黑阴影！）
- **金色**：仅作品牌强调（按钮、链接、关键数字、KPI），视觉占比 ≤ 5%
- **字体**：正文 sans / 标题 serif（Cormorant Garamond + 思源宋体 fallback）/ 数字 mono
- **阴影**：统一 `rgba(8, 22, 21, ...)`（带绿调），与墨绿背景协调
- **圆角**：大容器 22px / 小元素 12px / 徽章 999px
- **动效**：dropzone 内部 8s `liquidFloat` 浮动光晕；hover 金色辉光 + translateY(-1px)

## 关键工程决策
- **保留所有原始 CSS 变量名**：`--bg / --card / --text / --accent / --text-dim / --text-faint` 等，组件代码一行不用动
- **CSS 变量集中化**：色板 + 间距 + 阴影 + 圆角 + 字体全部走 `:root`，未来换主题改变量即可
- **backdrop-filter 兼容性**：所有都加了 `-webkit-backdrop-filter` 前缀（Safari）
- **响应式**：≤720px 时 topbar 垂直堆叠、grid-3 变单列、字号下调
